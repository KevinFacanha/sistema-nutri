# 🥗 Sistema Nutricionista


Sistema web para gestão de pacientes e acompanhamentos nutricionais, com filtros por tempo desde a última consulta e integração em tempo real com Supabase.


---


## 🚀 Funcionalidades


- Cadastro, edição e exclusão de pacientes
- Organização por profissionais
- Filtros por marcos de tempo (0–14, 15–29, 30–44, etc.)
- Identificação de pacientes vencidos
- Ação de acompanhamento (“Já contatei”)
- Busca por nome ou telefone


---


## 🛠️ Stack


- React + TypeScript
- Vite
- Tailwind CSS
- Supabase
